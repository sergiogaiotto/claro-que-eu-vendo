# Tendências de Mercado

## Objetivo
Fornecer análise atualizada de tendências do setor da empresa-alvo, posicionando o vendedor como consultor informado.

## Quando usar
- Vendedor pede "tendências", "panorama do setor", "o que está acontecendo no mercado de X"
- Preparação para demonstrar conhecimento setorial na reunião

## Workflow

1. **Identificar setor**: confirmar o setor exato (ex: "tecnologia" é genérico — refinar para "SaaS B2B", "fintech", etc.)
2. **Tendências macro**: 3-5 tendências que afetam o setor nos últimos 12 meses
3. **Impacto em PMEs**: como cada tendência afeta especificamente pequenas e médias empresas
4. **Oportunidades de venda**: para cada tendência, como o vendedor pode conectar com seu produto/serviço
5. **Dados e números**: estatísticas de mercado, crescimento, investimento

## Formato de saída

```
## Tendências — Setor [NOME DO SETOR]

### 1. [Nome da tendência]
Descrição da tendência com dados.
**Impacto em PMEs:** como afeta.
**Gancho de venda:** como usar na conversa.

### 2. [Nome da tendência]
...

### Dados do Setor
- Tamanho do mercado: R$ X
- Crescimento: X% a.a.
- Projeção: ...

### Fontes
[1] URL
```

## Regras
- Dados devem ter fonte e data. Não usar dados sem referência temporal.
- Focar em tendências acionáveis — que o vendedor possa mencionar na reunião.
- Conectar cada tendência com uma oportunidade prática de venda.
- Se o setor for muito amplo, pedir ao vendedor para especificar antes de pesquisar.
