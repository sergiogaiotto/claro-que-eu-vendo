# Objeção de Preço

## Objetivo
Ajudar o vendedor a lidar com objeções de preço de forma estratégica, reposicionando a conversa para valor.

## Quando usar
- Cliente diz que "está caro", "preço alto", "concorrente é mais barato"
- Vendedor precisa de argumentos para defender o preço
- Negociação travou por questão financeira

## Workflow

1. **Classificar a objeção**: preço absoluto, comparação com concorrente, falta de budget, ou tática de negociação
2. **Reposicionar**: mudar o frame de custo para investimento/retorno
3. **Fornecer técnicas**: 2-3 abordagens concretas para esta situação
4. **Script pronto**: frases prontas para usar

## Formato de saída

```
## Tratamento de Objeção de Preço

**Tipo:** [Preço absoluto | Comparação | Budget | Tática]

### Abordagem 1: [Nome da técnica]
**Conceito:** explicação em 1 linha
**Diga:** "frase pronta"

### Abordagem 2: [Nome da técnica]
**Conceito:** explicação em 1 linha
**Diga:** "frase pronta"

### O que NÃO fazer
- Não ...
- Não ...
```

## Técnicas disponíveis

### Isolamento
Separar o preço das outras variáveis: "Se o preço não fosse uma questão, esta seria a solução certa pra vocês?"

### Divisão
Dividir o valor pelo tempo ou pelo benefício: "São R$ X por dia / por colaborador / por lead gerado."

### Custo da inação
Quanto custa NÃO resolver o problema: "Quanto vocês estão perdendo por mês com [problema]?"

### Ancoragem de valor
Comparar com algo mais caro: "Contratar uma pessoa pra fazer isso custaria R$ X/mês. Nossa solução entrega o mesmo por R$ Y."

### Pacote ajustado
Propor escopo reduzido que caiba no budget: "Posso montar uma versão focada em [prioridade] que fica dentro do budget de vocês."

## Regras
- Nunca sugerir dar desconto como primeira opção.
- Sempre perguntar antes de argumentar — entender a objeção real.
- Se for tática de negociação, manter firmeza com elegância.
- Se for budget real, propor alternativas sem desvalorizar o produto.
